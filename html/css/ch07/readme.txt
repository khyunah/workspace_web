학습 목표 

flexbox에 대해 알아보자 

flex - 유연한, 2차원 레이아웃에서 잘 작동한다. 

///////////////////////////////////굉장히 중요/////////////////////////////////

규칙 1
flexbox 자식을 원하는 대로 움직이게 하기 위해서는 기본적으로 부모 요소에 명시를 한다. 
flexbox-container 로 만들어 주면 된다.

규칙 2
주축 ( main axis ), 교차축 ( cross axis ) 존재, 상황에 따라 달라진다. 
justify-content, justufy ( 현재 방향에 주축을 담당하는 속성 )
align-items ( 교차축을 담당하는 속성 )

기본적으로 주축은 수평, 교차축은 수직 으로 되어 있다. 
하지만 default기 때문에 변경 가능하다.

100vh : viewport Height 
스크린 마다 다르기때문에 화면을 꽉차게 해주는 속성이다. 

규칙 3
flex-wrap : 반응형에 따른 사이즈별 정렬을 알아서 해주는것
기본값은 nowrap
